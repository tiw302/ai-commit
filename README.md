# ai-commit 🤖

A tiny, humble AI-powered git commit message generator written in Go.  
*Made with ❤️ by a developer who's still learning and exploring!*

---

Hello! I'm just a beginner in the Go world, and I built this small tool to help myself (and maybe you!) write better, more consistent commit messages using AI. It's not perfect, but I'm trying my best to make it useful! 🌻

## ✨ Features
- **Zero-Config:** Works out of the box (it creates a default config for you!).
- **Custom Modes:** Use `--mode pro` for serious work or `--mode troll` for some fun with your team.
- **Easy Editing:** Not happy with the AI? Press `e` to edit the message in your favorite editor.
- **Smart Filtering:** Automatically ignores lock files and binaries to save your API tokens.
- **Built for Devs:** Clean, simple CLI with a colorful and friendly interface.
